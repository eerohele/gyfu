Define an ISO Schematron assertion.

of

An alias for an empty map.

If you want, you can use this to de-clutter pattern and rule declarations.

For example:

(rule "match/a/node" of
      (assert "The int value of <node> is > 0" "xs:int(.) gt 0"))
